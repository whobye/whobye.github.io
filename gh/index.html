GitHub 文件加速服务

支持 git clone, wget, curl 等命令行工具。

支持 raw.githubusercontent.com, gist.github.com, gist.githubusercontent.com 文件下载。

注意：不支持 SSH Key 方式 git clone 下载。

使用方法

命令行示例

git clone https://mirror.ghproxy.com/https://github.com/your_name/your_repo

下载示例

wget https://mirror.ghproxy.com/https://github.com/your_name/your_file
